项目简介

提供标准化的 Spring 分层架构代码 + MyBatis 的 CRUD 代码生成。
基于 FreeMarker 模板引擎实现，让代码生成更加灵活高效。

✨ 核心特性

🚀 快速生成

基于DDL语句一键生成完整的Java代码，包括Model、Service、Mapper等

📦 标准架构

遵循Spring分层架构设计，生成规范的代码结构

🔧 灵活配置

支持自定义包名、作者信息、Lombok、序列化等配置

💾 多数据库支持

支持MySQL、Oracle、PostgreSQL等主流数据库

🎨 代码高亮

集成Highlight.js，提供优雅的代码预览效果

📋 一键复制

支持快速复制生成的代码到剪贴板

🛠 技术栈

Spring Boot Thymeleaf FreeMarker MyBatis Druid JSqlParser Highlight.js

📝 使用示例

输入建表语句：

CREATE TABLE `article` ( `article_id` bigint(20) NOT NULL AUTO_INCREMENT COMMENT 'id', `article_title` varchar(512) NOT NULL COMMENT '标题', `article_author` varchar(128) NOT NULL COMMENT '作者', `article_content` text NOT NULL COMMENT '内容', `create_time` datetime DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间', PRIMARY KEY (`article_id`) ) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='文章主表';

点击"执行生成"按钮，即可自动生成对应的Java实体类、Service、Mapper和MyBatis XML配置文件。
