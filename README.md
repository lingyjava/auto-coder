## 📖 项目简介

AutoCoder 是一个基于 DDL 语句自动生成 Java 代码的工具，提供了现代化的 Web 界面，让代码生成更加简单高效。

## 🚀 快速开始

### 环境要求
- JDK 17+
- Maven 3.6+

### 启动应用
```bash
# 克隆项目
git clone https://github.com/lingyjava/auto-coder.git

# 进入项目目录
cd auto-coder

# 编译项目
mvn clean install

# 启动应用
mvn spring-boot:run -pl auto-coder-web
```

### 访问页面
启动后访问：`http://localhost:8081/`
